Automation Integration

Automation scripts can POST the test runs to the following rest endpoint.
Validations are done on STATUS & TEST_ENV.
The list of valid TEST_ENV are mentioned in the the application.properties config file.
http://HOSTNAME:HOST_PORT/rest/run/{TESTPLAN_ID}/{TESTCASE_ID}/{TEST_ENV}/{STATUS}/{COMMENTS}
